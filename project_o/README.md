### Описание проекта
Угадать загаданное компьютером число за мнимальное число попыток

:arrow_up:[к оглавлениею]()
### Какой кейс решаем
Нужно написать программу, которая угадывает число за минимальное число попыток

**Условия соревнования:**
- Компьютер загадывает целевое число от 0 до 100, нам нужно его угадать. Под "угадать" подразумевается "написать программу, которая угадывает число".
- Алгоритм учитывает информация о том, больше ли случайное число или меньше нужного нам.

**Метрика качества**
Результаты оцениваются по среднему количеству попыток при 1000 повторений

**Что практикуем**
Учимся писать хороший код на Python
